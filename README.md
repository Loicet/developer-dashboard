# Developer Dashboard

This is a personal developer dashboard built with **React** and **Tailwind CSS**.  
It displays my **GitHub profile**, **current weather**, and includes a **light/dark theme toggle** for improved user experience.

---

##  Features

- **GitHub Profile Section**  
  Fetches and displays GitHub avatar, username, followers, repos, and other profile details.

- **Weather Widget**  
  Shows real-time weather information using a weather API.

- **Dark / Light Theme Toggle**  
  Smooth theme switching using Tailwind’s dark mode.

- **Responsive UI**  
  Optimized for desktop and mobile screens.

---

##  Tech Stack

- React  
- Tailwind CSS  
- GitHub API  
- Weather API (e.g., OpenWeatherMap)

---

##  Installation & Setup

Clone the repository:

```bash
git clone https://github.com/Loicet/developer-dashboard.git
cd developer-dashboard

Install dependencies:

npm install


**Run the development server:

npm run dev


**Build for production:

npm run build
